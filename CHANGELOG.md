# Changelog

## [1.3.0] - 2026-06-14

### Added
- **多主题换肤**: 4 套全局主题色 — 默认高效、深夜沉浸、无压莫兰迪、多巴胺唤醒
- **主题切换器**: 导航栏 4 图标一键切换，丝滑过渡
- **四象限优先级色固定**: P0-P3 优先级色彩不随主题改变，保持全局一致

### Changed
- 全局 CSS 变量架构重构：`:root` + `[data-theme]` 驱动所有颜色
- 全部组件从硬编码 `stone-*`/`white`/`black` 迁移至 CSS 变量语义色（`bg-background`, `text-foreground`, `text-muted-foreground` 等）
- **优先级调色板极简化**：移除所有文字标签，纯色块+选中发光边框+Hover显示提示词（核心/长线/琐事/备忘），rounded-xl + 柔和阴影

## [1.2.0] - 2026-06-14

### Added
- **优先级系统**: P0-P3 四象限优先级（重要紧急/重要不紧急/紧急不重要/不重要不紧急）
- **优先级视觉**: 卡片左侧 4px 彩色指示条 + 小型优先级标签，避免大面积染色造成焦虑

## [1.1.0] - 2026-06-14

### Added
- **任务模式**: 支持创建一次性任务（区别于重复习惯），完成后自动标记为已完成
- **日历视图**: 月度打卡日历，可视化查看每天打卡情况，支持月份切换
- **里程碑庆祝弹窗**: 达成7/30/100天连续打卡时弹出专属庆祝动画
- **工作日频率**: 习惯新增"工作日"频率选项（周一至周五）

### Changed
- 重构 `calcStreak` 函数为共享工具模块 `src/lib/streak.ts`，消除3处重复代码
- 仪表盘区分任务与习惯的展示方式
- 创建表单支持选择任务/习惯类型

## [1.0.0] - 2026-06-14

### Added
- 初始版本：用户注册/登录（NextAuth + bcrypt）
- 习惯 CRUD（创建、编辑、删除、打卡）
- 数据统计页面（总打卡、连续天数、完成率、热力图）
- 成就系统（坚持新手3天、习惯养成者7天、高度自律30天、长期主义者100天）
- 响应式 UI（Tailwind CSS v4 + shadcn/ui）
